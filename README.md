# Monthly Operations Review Analysis

## Overview
This project analyzes the operational and financial performance of three store branches using sales data, operating costs, and customer feedback. The objective of the analysis is to evaluate branch profitability, compare performance visually, perform basic sentiment analysis on customer feedback, and provide business recommendations based on both quantitative and qualitative insights.

---

## Tasks Completed

### 1. Profitability Analysis
- Calculated Net Profit for each branch using:
  
Net Profit} = Total Sales - Operating Cost

- Calculated Profit Margin Percentage using:


{Profit Margin \%} = {{Net Profit}}/{{Total Sales}}*100


---

### 2. Data Visualization
- Created a bar chart comparing the Net Profit across all three branches using Plotly.

---

### 3. Sentiment Analysis
- Performed a simple keyword-based sentiment analysis on customer feedback.
- Ranked branches based on customer satisfaction and operational feedback.

---

### 4. Manager Recommendation
- Recommended which branch should receive a performance bonus.
- Identified which branch requires operational improvements based on financial and customer feedback analysis.

---

### 5. Bonus Challenge
- Calculated the updated Net Profit and Profit Margin for Branch C after reducing Operating Costs by 15%.

---

## Tools & Technologies Used
- Python

---

## Key Insights
- Branch B demonstrated the strongest overall performance due to positive customer feedback and the highest profit margin.
- Branch A showed stable profitability but requires queue management improvements.
- Branch C generated high sales but suffered from high operational costs and negative customer feedback, indicating the need for process optimization.
